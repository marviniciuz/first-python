# Operador de divisao

O uso do operado de divisao pode ser dividindo as duas variaveis no ```print``` ou adiciando a divisao de duas variaveis em uma vaiavel:

```python
number1 = 100
number2 = 4

print(number1 / number2)

25.0
```

```python
number1 = 200
number2 = 4

divisao = number1 / number2
print(divisao)

50.0
```
